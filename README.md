# my-next-app

Starter Next.js + TypeScript + Tailwind project ready for Vercel.

Setup

```powershell
cd my-next-app
npm install
npm run dev
```

Build

```powershell
npm run build
npm start
```

Deploy to Vercel

- Push this repo to GitHub.
- Connect the repository in the Vercel dashboard (or run `vercel` CLI).
